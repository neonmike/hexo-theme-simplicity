# Markdown语法示例

Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。本文将介绍Markdown的基本语法，帮助您更好地排版博客文章。

<!-- more -->

## 标题

Markdown支持六级标题，使用`#`符号表示：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 强调

*斜体文本* 或 _斜体文本_

**粗体文本** 或 __粗体文本__

***粗斜体文本*** 或 ___粗斜体文本___

## 列表

### 无序列表

- 项目1
- 项目2
  - 子项目2.1
  - 子项目2.2
- 项目3

### 有序列表

1. 第一项
2. 第二项
   1. 子项2.1
   2. 子项2.2
3. 第三项

## 代码

行内代码：`console.log('Hello World')`

代码块：

```javascript

  console.log('Hello World');
}

sayHello();
```
